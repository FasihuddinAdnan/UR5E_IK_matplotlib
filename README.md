# UR5e IK Solver — Built from Scratch

Complete Inverse Kinematics implementation 
for Universal Robots UR5e — no robotics 
libraries used!

## Features
- DH Parameter FK
- Jacobian-based IK
- Newton-Raphson solver
- Pick and Place animation

## How to run
pip install numpy matplotlib
python3 ur5e_pick_and_place.py
